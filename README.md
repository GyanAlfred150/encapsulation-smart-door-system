# Smart Door Access System for UMaT

## Description

This project implements a `Staff` class for a smart door access system.

## Features

-   Public attribute: `s_name`
-   Private attribute: `__access_code`
-   Uses `@property` and setter
-   Validates access code length (minimum 6 characters)
-   Displays staff information

Open the notebook and run all cells.
